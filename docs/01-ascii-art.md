# ASCII-ART
> **이미지 픽셀값**을 다양한 문자로 변환하여 콘솔에 출력하는 간단한 프로그램임.

## Execute
* 이미지, 비디오 두 가지 리소스에 대해 실행이 가능하며, 해당 경로는 직접 소스 코드에 입력해주어야 함.
* 실행법은 간단하며 아래와 같음.
```console
foo@bar:~$ python3 01-ascii-art.py
```
